Inštalácia

Oficialna inštalačná príručka: https://docs.djangoproject.com/en/1.7/topics/install/

Python 3.4

Vytvoríme a aktivujeme adresár envs3 pre lokálne závislosti

python3 -m venv envs3
(alternativne: pyvenv envs3)
source envs3/bin/activate
pip install django

Python 2.7

Vytvoríme a aktivujeme adresár envs pre lokálne závislosti

virtualenv envs
source envs/bin/activate
pip install django

Pokiaľ nemáte pip alebo virtualenv nainštalované, balíky python-pip, python-virtualenv sa nachádzajú v štandardných repozitároch Fedora, Ubuntu.

Prvá Django app

django-admin.py startproject konferencia

python manage.py migrate

python manage.py runserver

python manage.py startapp mailinglist

konferencia/ manage.py konferencia/ __init__.py settings.py urls.py wsgi.py

Modely

vim mailinglist/models.py

python manage.py makemigrations mailinglist
RAW SQL: python manage.py sqlmigrate mailinglist 0001

python manage.py migrate

from django.db import models class Subscriber(models.Model): email = models.CharField(max_length=255) subscribed_date = models.DateTimeField('Date subscribed') class MailingList(models.Model): name = models.CharField(max_length=50) subscriber = models.ManyToManyField(Subscriber)

Admin

python manage.py createsuperuser

vim mailinglist/admin.py

from mailinglist.models import Subscriber, MailingList admin.site.register(Subscriber) admin.site.register(MailingList)
